Listas

Las listas pueden ser ordenadas, desordenadas o de definición. Aquí tienes ejemplos:

Lista desordenada

Una lista desordenada muestra elementos sin un orden específico:

Por ejemplo, una lista de frutas:

  • Manzana
  • Banana
  • Naranja

Lista ordenada

Una lista ordenada muestra elementos en un orden específico:

Por ejemplo, una lista de pasos a seguir:

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

Lista de definición

Una lista de definición muestra términos y sus descripciones:
