Ejercicio de práctica de funciones

En este ejercicio, vamos a practicar el uso de funciones en JavaScript.

Abre la consola del navegador para ver los resultados.

1. Crea una función llamada saludoPersonalizado que reciba un nombre como argumento y muestre un saludo en la consola.

Podemos usar la función así: saludoPersonalizado("Juan") debería mostrar "Hola Juan, ¡bienvenido!" en la consola.

2. Crea una función llamada sumaDeDosNumeros que reciba dos números como argumentos y devuelva la suma de ambos. Puedes usar la función así: sumaDeDosNumeros(5, 10) debería devolver 15.

3. Crea una función llamada esPar que reciba un número como argumento y devuelva true si el número es par y false si es impar. Puedes usar la función así: esPar(4) debería devolver true.

4. Crea una función llamada mayorDeTres que reciba tres números como argumentos y devuelva el mayor de los tres. Puedes usar la función así: mayorDeTres(5, 10, 3) debería devolver 10.

5. Crea una función llamada ConversiónCelsiusAFahrenheit que reciba un número como argumento y devuelva la temperatura en grados Fahrenheit. Puedes usar la función así: ConversiónCelsiusAFahrenheit(0) debería devolver 32.
