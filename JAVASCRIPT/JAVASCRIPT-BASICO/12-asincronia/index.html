Asincronía en JavaScript

¿Qué es la Asincronía?

La asincronía es un concepto fundamental en JavaScript que permite la ejecución de código sin bloquear el hilo principal.

Esto es especialmente útil en operaciones que pueden tardar un tiempo indeterminado, como llamadas a APIs o lectura de archivos.

Ejemplos de Asincronía

  • Callbacks

    • Las funciones de callback son una forma de manejar la asincronía, pasando una función como argumento a otra función.

  • Eventos

    • Los eventos son otra forma de manejar la asincronía, permitiendo que el código responda a acciones del usuario o del sistema.

  • Promesas

    • Las promesas son un objeto que representa la eventual finalización (o falla) de una operación asíncrona y su valor resultante.

  • Async/Await

    • Async/Await es una forma más moderna y legible de manejar la asincronía en JavaScript, basada en promesas.
